get_next_utf8_character(resource  $handle) : null|boolean|string

Get the next UTF-8 character from a stream resource, advancing the file pointer

Get the next UTF-8 character (letter, rune, w/e) from a stream resource, advancing the file pointer. If the next byte in the stream is not a valid UTF-8 character or continuation, returns false and rewinds the file pointer. If attempting to read beyond EOF, returns NULL.

Parameters
